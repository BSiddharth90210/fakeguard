\# FakeGuard 🛡️

\### Fake News Detection System



A machine learning system that detects fake news using NLP and BERT.



\## Tech Stack

\- Python, FastAPI, PostgreSQL

\- Scikit-learn, XGBoost

\- HuggingFace Transformers (BERT)

\- LIME/SHAP Explainability
